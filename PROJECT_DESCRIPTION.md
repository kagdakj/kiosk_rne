# AI 음성 제어 스마트 키오스크

## 프로젝트 개요

연령별 맞춤형 UI와 AI 음성 인식 기술을 결합한 차세대 무인 주문 키오스크 시스템입니다. 사용자의 연령대를 자동으로 감지하여 최적화된 인터페이스를 제공하며, 자연어 음성 명령으로 메뉴 주문부터 장바구니 관리까지 모든 기능을 제어할 수 있습니다.

## 핵심 기능

### 1. 연령 기반 적응형 UI
- **자동 연령 감지**: face-api.js를 활용한 실시간 카메라 기반 연령 추정
- **4단계 맞춤 인터페이스**:
  - 어린이 (0-12세): 초대형 버튼, 큰 글씨
  - 청소년 (13-19세): 큰 버튼, 약간 큰 글씨
  - 성인 (20-64세): 표준 크기
  - 노인 (65세 이상): 초대형 버튼, 굵은 글씨
- **통일된 색상 시스템**: 모든 연령대에 일관된 브랜드 경험 제공

### 2. AI 음성 제어 시스템
- **자연어 처리**: "아메리카노 2잔 주세요" 같은 자연스러운 명령 인식
- **n8n 워크플로우 통합**: 음성 → AI 해석 → 자동 실행 파이프라인
- **7가지 액션 지원**:
  - 메뉴 추가 (옵션 포함)
  - 장바구니 관리 (비우기, 제거)
  - 주문 실행
  - 카테고리 탐색
  - 연령 모드 변경
  - 메뉴 상세 보기

### 3. 완전한 키오스크 기능
- 카테고리별 메뉴 탐색 (커피, 티, 디저트)
- 상세 옵션 선택 (사이즈, 당도, 얼음)
- 실시간 장바구니 관리
- 주문 및 결제 프로세스

## 기술 스택

### Frontend
- **HTML5/CSS3/JavaScript**: 순수 웹 기술 기반
- **Web Speech API**: 브라우저 내장 음성 인식
- **face-api.js**: TensorFlow 기반 얼굴 인식 및 연령 추정

### Backend/Integration
- **n8n**: 워크플로우 자동화 플랫폼
- **OpenAI/Claude**: AI 자연어 처리
- **Webhook**: 실시간 양방향 통신

### 디자인
- **반응형 디자인**: 다양한 화면 크기 지원
- **접근성 최적화**: 연령별 맞춤 UI
- **모던 UI/UX**: 그라데이션, 애니메이션, 시각적 피드백

## 주요 특징

✅ **완전 음성 제어**: 터치 없이 음성만으로 전체 주문 가능  
✅ **지능형 명령 해석**: AI가 자연어를 정확한 액션으로 변환  
✅ **접근성 우선**: 모든 연령대가 쉽게 사용 가능  
✅ **실시간 피드백**: 음성 인식 상태, AI 응답 메시지 표시  
✅ **확장 가능**: 새로운 메뉴, 액션 추가 용이  

## 사용 시나리오

1. **어린이 사용자**
   - 카메라가 자동으로 연령 감지 → 큰 버튼 UI 활성화
   - "쿠키 3개 주세요" → AI가 해석하여 자동 추가
   - 큰 글씨로 주문 내역 확인

2. **노인 사용자**
   - 초대형 버튼과 굵은 글씨로 편안한 사용
   - "커피 메뉴 보여줘" → 음성으로 카테고리 탐색
   - "장바구니 비워줘" → 복잡한 조작 없이 음성으로 관리

3. **일반 사용자**
   - "카페라떼 대 사이즈로 무가당, 얼음 많이" → 상세 옵션까지 한 번에 주문
   - 빠르고 정확한 주문 경험

## 향후 확장 가능성

- 다국어 지원 (영어, 중국어 등)
- 결제 시스템 통합
- 추천 시스템 (AI 기반 메뉴 추천)
- 고객 선호도 학습
- 음성 합성 (TTS)으로 응답 읽어주기
- 모바일 앱 연동

## 기술적 혁신

1. **AI + 음성 인식 융합**: 단순 명령어가 아닌 자연어 대화 지원
2. **실시간 연령 적응**: 사용자별 최적화된 경험 자동 제공
3. **확장 가능한 아키텍처**: n8n 워크플로우로 쉬운 기능 추가
4. **브라우저 기반**: 별도 앱 설치 없이 즉시 사용 가능
