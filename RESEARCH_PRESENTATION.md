# 2025학년도 융합과학탐구 발표회

## 주제: 소외 계층을 위한 AI 기반 적응형(Adaptive) 음성 제어 키오스크 연구

### I. 연구 동기 및 목적
 1. 연구 동기
  가. 디지털 소외 계층의 키오스크 사용 어려움 증가
   1) 고령층의 시각적/인지적 어려움
    가) 작은 글씨, 복잡한 UI, 낯선 용어로 인해 주문을 포기하거나 타인의 도움을 요청하는 사례가 빈번함
   2) 기존 키오스크의 획일적인 인터페이스
    가) 사용자의 특성(연령, 신체 조건)을 고려하지 않고 모든 사용자에게 동일한 화면을 제공하여 개인화된 경험 부족
  나. 터치 인터페이스의 한계와 음성 인식의 필요성
   1) 신체적 제약이 있는 사용자의 접근성 문제
    가) 휠체어 사용자나 손 떨림이 있는 노인의 경우 정확한 터치 조작이 어려워 음성과 같은 대안적 입력 방식이 절실함

 2. 연구 목적
  가. 연령별 맞춤형 UI 제공 시스템 구현
   1) 안면 인식을 통한 자동 연령대 판별 기술 적용
    가) 사용자 개입 없이 카메라만으로 연령을 추정하여, 노인에게는 큰 글씨와 단순한 화면을, 어린이에게는 눈높이에 맞는 UI를 즉각 제공
  나. AI 기반 자연어 음성 제어 시스템 구축
   1) 대화형 AI를 활용한 직관적인 주문 경험 제공
    가) 복잡한 메뉴 탐색 과정 없이 "아메리카노 따뜻한 거 한 잔 줘"와 같은 일상 언어로 주문부터 옵션 선택까지 가능하도록 구현

### II. 연구 방법 및 내용
 1. 연구 방법
  가. 하드웨어 및 개발 환경 구성
   1) 라즈베리파이(Raspberry Pi) 기반의 엣지 디바이스 활용
    가) 저전력, 소형화된 하드웨어에서도 구동 가능한 웹 기반(HTML5, JavaScript) 키오스크 프로토타입 제작
   2) 크로스 플랫폼 웹 기술 적용
    가) 별도의 앱 설치 없이 브라우저 환경에서 즉시 실행 가능하도록 설계하여 접근성 및 확장성 확보
  나. AI 모델 및 API 활용
   1) face-api.js를 이용한 클라이언트 측 연령 추정
    가) 서버로 이미지를 전송하지 않고 브라우저 내에서 실시간으로 얼굴을 감지하고 연령을 분석하여 개인정보 보호 및 속도 최적화
   2) n8n 워크플로우 자동화 도구와 OpenAI Whisper/LLM 연동
    가) 음성 데이터(STT) 처리 및 의도 파악(LLM)을 위한 백엔드 로직을 노코드/로우코드 툴인 n8n으로 구축하여 유연성 확보

 2. 연구 내용
  가. 적응형(Adaptive) UI 디자인 시스템 개발
   1) 4단계 연령 그룹(어린이, 청소년, 성인, 노인) 분류 알고리즘
    가) 감지된 연령에 따라 CSS 변수를 동적으로 제어하여 글자 크기, 버튼 크기, 레이아웃, 색상 대비를 실시간으로 변경
   2) 사용자 친화적 인터랙션 설계
    가) 카메라 오버레이를 통해 자신의 모습과 인식 상태를 직관적으로 보여주고, 음성 인식 상태를 시각적 파동으로 피드백 제공
  나. 음성 명령 처리 파이프라인 구축
   1) 브라우저 호환성을 고려한 오디오 캡처 방식 개선
    가) 라즈베리파이(Chromium)의 Web Speech API 호환성 문제를 해결하기 위해 MediaRecorder API로 오디오를 직접 녹음하여 서버로 전송하는 하이브리드 방식 고안
   2) AI 에이전트 기반의 키오스크 제어 로직
    가) 자연어 명령을 분석하여 '장바구니 추가', '옵션 변경', '메뉴 추천' 등의 구체적인 JSON 액션으로 변환하고 이를 자바스크립트 함수와 매핑하여 실행

### III. 연구 결과
 1. 연령 감지 및 UI 적응 성능
  가. 실시간 연령 추정 정확도 및 반응 속도
   1) 다양한 조명 환경에서의 얼굴 인식 테스트 결과
    가) 카메라 응시 후 약 1~2초 내에 연령대를 판별하고 UI가 즉시 변경됨을 확인하였으며, 오차 범위 내에서도 적절한 연령 그룹(성인/노인 등)으로 분류됨
  나. 사용자 경험(UX) 개선 효과
   1) 고령층 대상 가독성 비교
    가) 기본 모드 대비 '시니어 모드' 진입 시 버튼 크기가 1.5배 확대되고 명도 대비가 높아져 메뉴 식별이 용이해짐

 2. AI 음성 제어 시스템 동작
  가. 음성 인식률 및 의도 파악 정확도
   1) 복합 명령 처리 성공률
    가) "라떼 덜 달게 해서 한 잔이랑 쿠키 줘"와 같은 복합 주문을 Whisper 모델이 정확히 텍스트로 변환하고, LLM이 옵션(당도, 수량)까지 정확히 구조화된 데이터로 추출함
  나. 라즈베리파이 환경에서의 안정성
   1) HTTPS 및 오디오 전송 최적화 결과
    가) 저사양 ARM 환경에서도 네트워크 오류 없이 음성 파일이 전송되고, 약 2~3초 내에 AI 응답을 받아 주문이 처리되는 안정적인 성능을 확인함

### IV. 결론 및 제언
 1. 결론
  가. 소외 계층을 포용하는 기술적 대안 제시
   1) 기술이 장벽이 아닌 도구가 되는 인터페이스 구현
    가) 연령 감지와 음성 인식을 결합하여 디지털 기기 사용에 어려움을 겪는 계층에게 실질적인 도움을 줄 수 있는 가능성을 입증함
  나. 저비용 고효율의 AI 키오스크 솔루션 확보
   1) 고가의 전용 하드웨어 없이 웹 기술과 API만으로 구현 가능
    가) 소상공인도 쉽게 도입할 수 있는 경제적이고 접근성 높은 스마트 키오스크 모델을 제시함

 2. 제언
  가. 시스템 고도화 방향
   1) 다국어 지원 및 추천 알고리즘 도입
    가) 외국인 관광객을 위한 실시간 통역 기능과 사용자의 이전 주문 이력을 학습하여 메뉴를 추천하는 개인화 AI 도입 필요
  나. 하드웨어 최적화 및 보안 강화
   1) 엣지 AI 가속기(NPU) 활용 연구
    가) 클라우드 의존도를 낮추고 개인정보 보호를 강화하기 위해, 음성 인식과 자연어 처리를 로컬 디바이스(On-device AI)에서 수행하는 방향으로 발전 필요
